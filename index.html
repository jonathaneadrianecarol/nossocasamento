Vai chover?

Temos um plano B coberto no próprio espaço. Mantemos todos informados se houver mudança.

Posso levar acompanhante?

Considere o convite/RSVP. Se tiver dúvida, fale conosco antes.

Qual o melhor horário para chegar?

Chegue por volta de 16:00 para estacionar com calma e aproveitar as boas-vindas.

Tem dress code obrigatório?

Não. A ideia é estar confortável. Recomendamos “praia chique” e cores claras.
